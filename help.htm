CheckHashGUI

Ett verktyg för att jämföra hash:en för en fil med den officiella hash:en av filen.

Användning:

  1. Rensa inmatnings/utmatningsfönstret om någon text visas: Klicka på "Clear".
  2. Bläddra till den nedladdade filen du vill kontrollera: Klicka på "Browse".
  3. Kopiera den officiella hash:en från nedladdningssidan till inmatnings/utmatningsfönstret.
  4. Klicka på "Check".
  5. Hash:en av filen beräknas och jämförs med hash:en du matade in. Båda hash:arna visas, liksom resultatet av jämförelsen.

CheckHashGUI

A tool to compare the hash of a file with the official hash of the file.

How to use:

  1. Clear the input/output window, if any text is displayed: Click on "Clear".
  2. Browse to the downloaded file you want to check: Click on "Browse".
  3. Copy the official hash from the download page to the input/output window.
  4. Click on "Check".
  5. The hash of the file is computed and compared with the hash you entered. Both hashes are displayed, as well as the result of the comparison.
